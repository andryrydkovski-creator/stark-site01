# Сайт Костоправа Старка

Профессиональный лендинг для мастера мануальной коррекции.

## Как запустить этот проект у себя (для разработчиков)

1. Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).
2. Скачайте файлы проекта.
3. В терминале (в папке проекта) выполните:
   ```bash
   npm install
   ```
4. Для запуска в режиме разработки:
   ```bash
   npm run dev
   ```
5. Для сборки готового сайта (папка `dist`):
   ```bash
   npm run build
   ```

## Как опубликовать сайт БЕСПЛАТНО (для новичков)

Самый простой способ — использовать сервис **Vercel**:

1. Зарегистрируйтесь на [Vercel.com](https://vercel.com).
2. Установите Vercel CLI на компьютер или подключите свой аккаунт GitHub.
3. Если используете GitHub:
   - Создайте новый репозиторий.
   - Загрузите туда все файлы проекта (кроме папки `node_modules`).
   - В Vercel нажмите "Add New" -> "Project" и выберите этот репозиторий.
   - Нажмите "Deploy". Готово!
